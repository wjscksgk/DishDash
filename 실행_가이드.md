# Dish Dash 실행 가이드

Dish Dash는 온디바이스 Gemma LLM으로 한국 배달 메뉴를 생성하고, 열 가지 메뉴가 경주하는 Flutter 앱입니다.
LLM 모델 파일이 없어도 데모 모드(고정 메뉴)로 앱은 정상 실행됩니다.

---

## 제출 범위

| 플랫폼 | 제출 여부 | 비고 |
|--------|-----------|------|
| Android | ✅ APK 제출 | 데모 모드만 검증됨 |
| Windows | ✅ zip 제출 | AI 모드 포함 |
| iOS | ❌ 미제출 | 무료 Apple 개발자 계정으로는 타인 기기에 설치 가능한 배포 빌드 생성 불가 |

---

## Android

### 요구 사항
- Android 7.0 (API 24) 이상, arm64-v8a 기기

### 앱 설치
`app-release.apk` 파일을 기기로 옮겨 실행하면 설치됩니다.

### 참고
Android 실기기가 없어 LLM 기능을 검증하지 못했습니다. **데모 모드(고정 메뉴)로만 동작이 확인된 빌드**입니다.

---

## Windows

### 요구 사항
- Windows 10 64비트 이상

### 앱 실행
zip 파일 압축을 해제하고 폴더 안의 `dish_dash.exe`를 실행합니다.
exe 파일을 폴더 밖으로 꺼내면 실행되지 않습니다.

### 모델 설치 (선택)
AI 모드로 실행하려면 아래 경로에 `models` 폴더를 만들고 모델 파일을 넣습니다.

```
C:\Users\<사용자명>\Documents\models\gemma-4-e2b-it.litertlm
```

예시:
```
C:\Users\홍길동\Documents\models\gemma-4-e2b-it.litertlm
```

모델 파일은 아래 링크에서 다운로드할 수 있습니다 (무료 계정 필요).
- Hugging Face: https://huggingface.co/litert-community/gemma-4-E2B-it-litert-lm
- Kaggle: https://www.kaggle.com/models/google/gemma-4 → LiteRT 탭 → `gemma-4-e2b-it.litertlm`

---

## 데모 모드 vs AI 모드

| 구분 | 데모 모드 | AI 모드 |
|------|-----------|---------|
| 모델 파일 | 불필요 | 필요 |
| 메뉴 생성 | 고정 메뉴 사용 | Gemma LLM이 실시간 생성 |
| 우승 코멘트 | 기본 문구 | AI가 생성한 감각적인 한 마디 |
| 상단 표시 | `DEMO MODE` | `AI READY · Gemma 4 E2B` |
